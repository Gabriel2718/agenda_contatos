# Agenda de Contatos

Um projeto Flutter.

## Descrição

Este projeto consiste em uma agenda de contatos, na qual o usuário pode editar o nome e-mail, telefone e atribuir uma 
foto de perfil para cada contato.

## Bibliotecas Utilizadas

- cupertino_icons v1.0.8
- sqflite v2.4.2
- url_launcher v6.3.2
- image_picker v1.1.2
